# Form Validation (JavaScript)

This project is a simple client-side form validation system built using HTML, CSS, and JavaScript.

## Purpose
This project was created as part of revising core JavaScript fundamentals and strengthening understanding of DOM manipulation and input validation.

## Features
- Required field validation
- Email validation
- Prevents form submission on invalid data

## Tech Stack
- HTML
- CSS
- JavaScript

## What I Focused On
- Writing validation logic without relying on libraries
- Handling multiple error states
- Improving user feedback through UI changes

## Status
Revision project – focused on improving fundamentals
